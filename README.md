# Java Cafe

A full-stack bakery menu application built with Java, React, PostgreSQL, and Tailwind CSS.

## Project Overview

- Converting from ice cream P0 project to bakery menu
- 3 pages: main, menu, place order
- Products: cake, croissants, cupcakes, cookies

## Tech Stack

- **Backend**: Java
- **Frontend**: React
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

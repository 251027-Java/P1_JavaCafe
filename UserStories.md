User Stories:

Customer Stories:

1. View Products
    As a customer, I want to view all available bakery products on the home page, so that I can see what's available.

2. Browse by Category
    As a customer, I want to filter products by category (Cakes, Croissants, Cupcakes, Cookies), so that I can quickly find the type of item I'm looking for.

3. View Product Details
    As a customer, I want to see product descriptions, prices, and images, so that I can make informed purchasing decisions.

4. Check Availability
    As a customer, I want to see if a product is "Sold Out", so that I don't try to order unavailable items.

5. Add to Cart
    As a customer, I want to add products to my cart, so that I can purchase multiple items in one order.

6. Adjust Quantities
    As a customer, I want to increase or decrease item quantities in my cart, so that I can order the exact amount I need.

7. View Cart Total
    As a customer, I want to see the subtotal, tax, and total cost in real-time, so that I know how much I'll pay.

8. Place Order
    As a customer, I want to enter my contact information and place an order, so that the bakery can prepare my items.

9. Order Confirmation
    As a customer, I want to receive an order confirmation with order number and details, so that I have proof of my purchase.
    

Staff/Admin Stories:

10. Track Inventory
    As a staff member, I want inventory to automatically decrease when orders are placed, so that stock levels are accurate.

11. View Orders
    As a staff member, I want to view customer orders with timestamps and status, so that I can fulfill them.

12. Manage Stock
    As a manager, I want to update product inventory levels, so that I can restock items.
